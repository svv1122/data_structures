Data Structures
===============

This project aims to provide a detailed introduction to as many data structures as possible.
Each data structure is implemented in **C**, with a focus on clarity and completeness.

Note: This repository includes **only the implementation code**. Test cases, benchmarks, and usage examples are **not** included at this stage.

Project Goals
-------------

- Learn and implement data structures as completely as possible
- Understand how they are implemented in C
- Explore more complete and formal definitions
